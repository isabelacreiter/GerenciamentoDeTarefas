# 📌 Gerenciamento de Tarefas

Sistema web para gerenciamento de tarefas desenvolvido com **ASP.NET Core MVC** e **Entity Framework Core**, permitindo organizar atividades por categoria, prazo e status.

---

## 🚀 Funcionalidades

- ✅ Cadastro de novas tarefas
- 📋 Listagem de tarefas
- 🏷️ Organização por categorias
- 📅 Controle de prazo (vencimento)
- ✔️ Marcar tarefa como concluída
- 🗑️ Exclusão de tarefas concluídas
- 🔍 Filtros por:
  - Categoria
  - Prazo
  - Situação (status)
  - Data de criação
- ⚠️ Destaque visual para tarefas atrasadas

---

## 🖥️ Tecnologias Utilizadas

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server / LocalDB
- Bootstrap
- C#
- Razor Pages (Views)

## 🧠 Modelo de Dados

### Tarefa

| Campo | Tipo | Descrição |
|--------|--------|------------|
Id | int | Identificador |
Titulo | string | Título da tarefa |
Descricao | string | Detalhes |
DataCriacao | DateTime | Data de criação |
DataVencimento | DateTime? | Prazo |
CategoriaId | string | Categoria |
StatusId | int | Status (Pendente/Concluído) |
